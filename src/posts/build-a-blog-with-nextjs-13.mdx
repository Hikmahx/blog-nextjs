In this tutorial, we will explore how to build a blog using Next.js version 13. Next.js is a powerful React framework that offers server-side rendering, automatic code splitting, and an intuitive page-based routing system, making it ideal for creating performant and SEO-friendly web applications.

### Getting Started

To begin, make sure you have Node.js and npm (Node Package Manager) installed on your development machine. You can install Next.js globally using npm:

<SyntaxHighlighter language="bash">
  {`
    npm install -g next
  `}
</SyntaxHighlighter>

### Setting Up Your Next.js Project

Create a new Next.js project by running the following commands:

<SyntaxHighlighter language="bash">
  {`
    npx create-next-app@13 my-blog
    cd my-blog
  `}
</SyntaxHighlighter>

## Creating Pages

Next.js uses a pages-based routing system. Create a new directory called `pages` in the root of your project. Inside this directory, create a file named `index.js` for the homepage and `blog.js` for the blog page.

### Homepage (`index.js`)

<SyntaxHighlighter language="jsx">
  {`
    import Link from "next/link";

    export default function Home() {
      return (
        <div>
          <h1>Welcome to My Blog</h1>
          <p>Explore our latest posts:</p>
          <ul>
            <li>
              <Link href="/blog/first-post">First Post</Link>
            </li>
            <li>
              <Link href="/blog/second-post">Second Post</Link>
            </li>
            {/* Add more links for other posts */}
          </ul>
        </div>
      );
    }

`}

</SyntaxHighlighter>

<SyntaxHighlighter language="jsx">
  {`
    // pages/blog.js
    export default function Blog() {
      return (
        <div>
          <h1>Blog</h1>
          {/* Render list of blog posts here */}
        </div>
      );
    }
  `}
</SyntaxHighlighter>

## Fetching Blog Data

<SyntaxHighlighter language="jsx">
{`
  // pages/blog.js
  import { useEffect, useState } from "react";
  import Link from "next/link";
  import posts from "../data/posts.json";

  export default function Blog() {
    const [blogPosts, setBlogPosts] = useState([]);

    useEffect(() => {
      setBlogPosts(posts);
    }, []);

    return (
      <div>
        <h1>Blog</h1>
        <ul>
          {blogPosts.map((post) => (
            <li key={post.id}>
              <Link href={\`/blog/\${post.id}\`}>
                <a>{post.title}</a>
              </Link>
            </li>
          ))}
        </ul>
      </div>
    );
  }
`}
</SyntaxHighlighter>
